# dapp-kit-core

A framework-agnostic toolkit for building decentralized applications on [Sui](https://sui.io).

`@mysten/dapp-kit-core` provides an action-based API for wallet operations, automatic state
management via [nanostores](https://github.com/nanostores/nanostores), and built-in
[Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) for wallet
connection UI. It can be used standalone with vanilla JavaScript, or as the foundation for
framework-specific bindings like
[`@mysten/dapp-kit-react`](https://www.npmjs.com/package/@mysten/dapp-kit-react).

## Installation

```sh
npm i @mysten/dapp-kit-core @mysten/sui
```

## Documentation

See the [dApp Kit docs](https://sdk.mystenlabs.com/sui/dapp-kit) for full usage instructions,
guides, and API reference.
